# 宝石16タイプ診断

16の問いから、あなたに宿る原石のかたちを見つける宝石診断サイト。

ニックネームと生年月日を入力し、12問の質問に答えると、あなたの性格に合った宝石が診断されます。  
生年月日から誕生石も取得し、メイン宝石との組み合わせメッセージも生成します。

## 特徴

- オリジナル4軸 × 16タイプ × 16種の宝石
- 生年月日から誕生石を取得し、メイン宝石との組み合わせメッセージを生成
- 白基調 × ゴールドアクセントの上品なデザイン
- レスポンシブ対応（スマホ・PC両対応）
- 外部ライブラリ不要（Google Fontsのみ）

## 動かし方

`index.html` をブラウザで開くだけで動きます。

ローカルサーバーで起動する場合：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```
